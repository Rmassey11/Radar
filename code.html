The Code for Radar


                    //Get values from page
                    function getPalindrome(){
                        document.getElementById("alert").classList.add("invisible");
                        let pageValues = document.getElementById("userPalindrome").value;
                        let returnObj = revPalindrome(pageValues);
                        displayPalindrome(returnObj);
                        }

                    //Reverse the values
                    function revPalindrome(pageValues){
                        pageValues = pageValues.toLowerCase();
                        let regex = /[^a-z0-9]/gi;
                        pageValues = pageValues.replace(regex,"");
                        let revPalindrome = [];
                        let returnObj = {};
                        for(let index = pageValues.length - 1; index >=0; index--){
                            revPalindrome += pageValues[index];
                        }
                        if(revPalindrome == pageValues){
                            returnObj.msg = "Well done! You enetered a Palindrome"
                        }
                        else{
                            returnObj.msg = "Sorry! You did not enter a palindrome."
                        }
                        returnObj.reversed = revPalindrome;
                        return returnObj;
                    }

                    //Display success and reversed values
                    function displayPalindrome(returnObj){
                        document.getElementById("alertHeader").innerHTML = returnObj.msg;
                        document.getElementById("msg").innerHTML = `Your Phrase reversed is: ${returnObj.reversed}`;
                        document.getElementById("alert").classList.remove("invisible");
                    }

The Code is structured in three functions.

getPalindrome

The getPalindrome function retrieves the entered word or phrase from the page for use in our other functions. First we set our alert window to invisible to make sure it is not seen until the app runs. We then declare a variable pageValues and set it to the values entered by the user. Next we declare another variable returnObj and set it to the value returned by the revPalindrome function. Finally we pass the returnObj, which will be an Object at this point, to our displayPalindrome function.

revPalindrome

The revPalindrome function takes the pageValues gathered by our getPalindrome function, converts all letters to lower case, removes all special characters and spaces, reverses the remaining characters, and finally checks to see if the entered value is a palindrome. To start we take our pageValues and use the toLowerCase method to make all letters lower case. Next by using a Regular Expression we confirm that all letters or numbers are lower case or numbers and remove all other characters. Then we declare the revPalindrome variable and set it to an empty array and another variable returnObj which is set to an empty object. Next we run a for loop through our pageValues string starting at the end and looping until we reach the beginning. At each iteration of our loop we add the value at each index to our revPalindrome array. Finally we check if our new revPalindrome array is equal to the pageValues array. If it is we set our returnObj.msg to a success message. If not then returnObj.msg is set to a failure message. We then set returnObj.reversed to equal our new revPalindrome array and return the returnObj variable to getPalindrome which is an object containing our revPalindrome array as well as the correct message depending on the results.

displayPalindrome

The displayPalindrome function takes the object returned from the revPalindrome function and displays both the success/fail message and the reversed value contained within it. First we target our alertHeader from our app.html page and using innerHTML we set it to our success/fail message attatched to the returnObj variable. Next we target the msg Id from the app.html page which is where our message content will go. Using innerHTML we set that to display our reversed phrase. Finally we target the entire alert system from app.html and remove the "invisible" class attatched at the beginning so that the content can be displayed.
